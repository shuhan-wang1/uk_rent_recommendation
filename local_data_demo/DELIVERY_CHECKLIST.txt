## 方式 1️⃣ - 直接使用单个工具（最简单）
代码行数: 5 行
```python
from core.tools import search_properties_tool
result = await search_properties_tool.execute(
    location="UCL",
    max_budget=1500
)
```
用途: 简单任务，单一操作

---

## 方式 2️⃣ - 使用工具注册表（中等难度）
代码行数: 10 行
```python
registry = create_tool_registry()
result1 = await registry.execute_tool('search_properties', ...)
result2 = await registry.execute_tool('calculate_commute', ...)
```
用途: 多步骤任务，明确的执行顺序

---

## 方式 3️⃣ - 完全自动化 Agent（最强大）⭐⭐⭐
代码行数: 5 行
```python
agent = Agent(registry, call_ollama)
result = await agent.run(
    "Find me a flat near UCL with less than 30 min commute to King's College"
)
```
用途: 复杂任务，无需预定义流程，**完全自动化**
